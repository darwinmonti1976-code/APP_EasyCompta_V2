# Captures d'écran — EasyCompta

## Comment capturer (iPhone 6.7" = 1290×2796 px)

1. Ouvre chaque fichier HTML dans **Google Chrome**
2. Ouvre les DevTools → Device Toolbar (Ctrl+Shift+M)
3. Sélectionne **"Responsive"** et entre **430 × 932**
4. Règle le DPR sur **3** (Device pixel ratio)
5. Clic droit sur la page → **Capture screenshot** → enregistre en PNG
6. Le fichier fait exactement 1290×2796 px ✓

- **iPhone 6.7"** : 1290×2796 px (430×932 @ 3x) — screenshots ci-dessus
- **iPhone 6.5"** : 1242×2688 px (414×896 @ 3x) — recapturer à 414×896
- **Android** : 1080×1920 px (360×640 @ 3x) — recapturer à 360×640
